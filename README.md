# スクスタDeep Dive!

## 本サイトについて

スクスタDeep Dive！は、ラブライブ！スクールアイドルフェスティバルALL STARS（スクスタ）におけるライブの仕様について、浦の星パソコン部（@devsifmatch）が検証した内容をまとめるWebサイトです。
